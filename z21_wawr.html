Pojęcie Znaczenie
Co znaczy skrót ID? ID to unikalny identyfikator przypisany do jednego elementu na stronie.
Jaki jest cel stosowania ID? ID jest używane do jednoznacznego identyfikowania i stylowania konkretnego elementu HTML na stronie.
Jakie jest znaczenie znaku #? Znak # jest używany w CSS do wskazania, że chcemy stylować element z określonym ID.
Jak następuje wywołanie ID (wraz z wytłumaczeniem)? ID wywołuje się przez przypisanie atrybutu id do elementu w HTML oraz odniesienie się do tego ID w CSS poprzez znak #, np. <p id="example">.
Uwagi na temat stosowania ID (trzy konieczne wyliczenia)
  1. ID musi być unikatowe dla każdego elementu na stronie.
  2. ID powinno być stosowane tylko wtedy, gdy jest pewność, że dany element pojawi się na stronie tylko raz.
  3. Jeżeli ten sam styl ma być przypisany do wielu elementów, lepiej użyć klasy zamiast ID.
