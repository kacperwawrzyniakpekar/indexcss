Zadanie 9: Opis stylu zewnętrznego

1. Składnia teoretyczna


selector {
    property: value;
}

W składni CSS:

  • selector oznacza element HTML, do którego stosowany jest styl.
  • property to właściwość CSS, którą chcesz ustawić.
  • value to wartość przypisywana właściwości.

2. Gdzie wstawiany jest styl zewnętrzny, jaki znacznik jest za to odpowiedzialny oraz ile razy może on występować w dokumencie, jako podłączenie stylu CSS?

Styl zewnętrzny jest wstawiany w sekcji <head> dokumentu HTML. Znacznik odpowiedzialny za to to <link>. Może występować jeden raz w dokumencie HTML jako podłączenie stylu CSS.

3. Jak jest najprostsza forma wstawiania stylu zewnętrznego


<link rel="stylesheet" href="styles.css">

W tym przykładzie rel określa relację między dokumentem a zewnętrznym plikiem CSS, a href wskazuje lokalizację pliku stylów.

4. Ile arkuszy styli może być dołączanych do jednego dokumentu HTML oraz jaka zasada musi być zachowana przy większej ilości dołączanych styli?

Można dołączyć wiele arkuszy styli do jednego dokumentu HTML. W przypadku większej liczby arkuszy styli, należy zachować kolejność, w jakiej są one dołączane, ponieważ style z późniejszych arkuszy mogą nadpisywać style z wcześniejszych.

5. Co się dzieje w przypadku konfliktów? Przez konflikt rozumiemy, gdy te same style są definiowane w różnych arkuszach styli podłączone do tego samego arkusza.

W przypadku konfliktów, ostateczny styl jest określany przez specyfikację CSS, czyli zasadę kaskadowania. Style z późniejszych arkuszy lub bardziej specyficzne selektory nadpiszą style z wcześniejszych arkuszy.

6. Jak są tworzone i edytowane pliki arkuszy z użyciem edytora tekstowego?

Pliki arkuszy stylów mogą być tworzone i edytowane w dowolnym edytorze tekstowym, takim jak Notepad++ czy Sublime Text. Wystarczy zapisać plik z rozszerzeniem .css i wprowadzać zmiany w składni CSS.

7. Jak są tworzone i edytowane pliki arkuszy z użyciem specjalnego edytora CSS?

Specjalistyczne edytory CSS, takie jak Adobe Brackets czy Visual Studio Code, oferują funkcje ułatwiające pisanie CSS, takie jak podpowiedzi składniowe, automatyczne uzupełnianie i wizualizacje. Pliki tworzy się podobnie jak w edytorze tekstowym, ale edytory te oferują dodatkowe narzędzia do zarządzania stylami.

8. Jakie rozszerzenie posiada plik będący zewnętrznym arkuszem styli?

Pliki zewnętrznych arkuszy styli mają rozszerzenie .css.

9. Jakie są zalety zewnętrznego arkusza styli?

  • Łatwość zarządzania - Centralne miejsce na wszystkie style pozwala na łatwiejszą edycję i utrzymanie kodu.
  • Lepsza organizacja - Oddziela treść od stylów, co ułatwia pracę nad kodem HTML i CSS niezależnie.

10. Uwagi na temat pisania w CSS

  • Konsystencja - Stosuj spójną konwencję nazewnictwa i formatowania, aby kod był czytelny.
  • Optymalizacja - Unikaj nadmiernego użycia zagnieżdżonych selektorów, aby zwiększyć wydajność i przejrzystość kodu.
  • Komentarze - Dodawaj komentarze, aby wyjaśnić trudniejsze fragmenty kodu lub zamierzenia.
